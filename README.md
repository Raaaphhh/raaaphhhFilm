# 🎬 Portfolio Vidéo - raaaphhhFilm (C#)

Une application web moderne développée en **C# /.NET Core** pour centraliser et présenter mes créations vidéo artistiques diffusées sur Youtube.

## Aperçu du Projet
Ce projet sert de vitrine pour mes travaux de montage, et
réalisation. L'application permet aux visiteurs de 
naviguer entre différentes vidéo ainsi que de les visionner
directement via une interface fluide.
Il existe une partie pour les utilisateurs privés qui leurs permets de taper un code pour avoir accès a des vidéos privées.

### Fonctionnalités principales :
* **Galerie Dynamique :** Affichage des vidéos avec miniatures et titres.
* **Lecteur Intégré :** Support des lecteurs YouTube.

## Stack Technique
* **Backend :** ASP.NET Core 8.0 (C#)
* **Frontend :** Razor Pages / Bootstrap 5
* **Données** Crawler python recupérant mes vidéos directement sur ma chaine YTB.
